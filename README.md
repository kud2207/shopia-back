Welcome to the SHOPIA project, the AI-powered application that is revolutionising e-commerce in Africa

## Getting Started

First, rename .env.local.example to .env.local and edit environment variable

Secondly, intall dependencies: 
```bash
npm install
# or
yarn install
# or
pnpm install
```

And, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3001](http://localhost:3001) with your browser to see the result.

You can start editing the page by modifying `pages/index.js`. The page auto-updates as you edit the file.

[API routes](https://nextjs.org/docs/api-routes/introduction) can be accessed on [http://localhost:3001/api/hello](http://localhost:3001/api/hello). This endpoint can be edited in `src/pages/api/hello.js`.

The `src/pages/api` directory is mapped to `/api/*`. Files in this directory are treated as [API routes](https://nextjs.org/docs/api-routes/introduction) instead of React pages.
